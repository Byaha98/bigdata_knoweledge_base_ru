# Добро пожаловать в базу знаний для BigData специалистов на русском языке!

1. Если хотите просто работать с Obsidian частью базы, то , скачайте Obsidian https://obsidian.md/download
2. Затем склонируйте репозиторий
3. Откройте Obsidian, в нем откройте папку с репозиторием и сделайте ее как vault. Основной контент базы лежит в папке content
4. В диалоговом окне подтвердите, что доверяете авторам.


Если вас интересует развертывание форка моей базы знаний в том числе с деплоем сайта, то вам понадобятся NodeJS, NPM, Obsidian, Quartz
